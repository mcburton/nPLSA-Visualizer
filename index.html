What is this?

This site is a tool for analyzing the nPLSA topic modeling process. nPLSA or nonparametric Probabilistic Latent Semantic Analysis is a forthcoming topic modeling technique by Jian Tang, Ming Zhang, and Qiaozhu Mei. The nPLSA process is unique from other more popular topic modeling techniques like Probabilistic Latent Semantic Analysis or Latent Dirichlet Allocation (LDA) because it does not require you to specify the number of topics a priori. nPLSA infers the number of topics through an iterative process that creates new topics from documents in the corpus. For more information about nPLSA please read the paper on Arxive.org [link to be posted].

The purpose of this tool is to being providing insight into the iterative process by visualizing the emergence of new topics and their relations at each step of the modeling process. Currently the tool supports two kinds of interactive visualizations:

  • A stacked area chart of topic size (measured in # of tokens) over time (in steps)
  • An animated Voronoi scatter plot of topic’s dimensionally reduced relationships over time (in steps)

The data for these visualizations is generated from a python script, topic-viz.py. The script ingests the model output from the java implementation of nPLSA, Clean.ParameterFreeNonParametricPLSA. The application produces several files at each iterative step:

  • nplsa.model.NUM - the topic/word distributions per step(NUM)
  • nplsa.topicSize.NUM- the number of tokens per topic per step(NUM)
  • nplsa.topicMixture.NUM - the topic/document distributions per step(NUM)

The python script consumes the nplsa.model.num and nplsa.topicSize.num files and generates JSON files for the visualization. The specific processing details are covered on each visualization’s page. They can be visited by clicking the images to the left or via the navigation bar at the top of the page.
